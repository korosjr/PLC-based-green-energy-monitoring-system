# PLC-based-green-energy-monitoring-system
Real-time monitoring and data acquisition system for solar panels' installations using Siemens S7-1200 PLCs. The system tracks KPIs, calculates energy production metrics, manages alarms and provides visualization through an HMI interface. 

# Tech Stack
PLC Platform: Siemens S7-1200
IDE: TIA Portal 
Programming Languages: Structured Control Language(SCL) & Ladder Logic(LAD)
HMI: Siemens Comfort Panel (KTP700)
Simulation: PLCSIM Advanced
Version Control: Github
